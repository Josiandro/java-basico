Tipos primitivos

Declaração no Visualg
    
var 
    idade : Inteiro 
    sal : Real
    letra : Caractere 
    casado : Logico

inicio
    idade <- 3
    sal <- 1825.54
    letra <- "G"
    casado <- falso

Declaração no Java

1ª Forma:

int idade = 3;
float sal = 1825.54f;
char letra = 'G';
boolean casado = false;

2ª Forma (typecast): Vai forçar o tipo do valor

int idade = (int) 3;
float sal = (float) 1825.54;
char letra = (char) 'G';
boolean casado = (boolean) false;

3ª Forma (Wrapper Class): Cria um objeto da classe Wrapper e passa um valor

Integer idade = new Integer(3);
Float sal = new Float(1825.54);
Character letra = new Character('G');
Boolean casado = new Boolean(false);
